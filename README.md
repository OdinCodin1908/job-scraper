# Job Scraper

A Python script that searches for live job listings using the Adzuna API and exports results to a CSV file.

## Features
- Search jobs by keyword and location
- Displays title, company, location, salary and URL
- Saves results to a CSV file

## Tech Stack
Python, Requests, Adzuna API
